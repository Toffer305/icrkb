Software designed to aid deployment or installation of IC Realtime hardware. These include softwares which will discover IP devices, configure parameters, and necessary packaged required for normal user operation of certain hardware devices.

ICIP Config Tool

The ICIP series network discovery and configuration utility. This software is compatible with ICRealtime Recorder devices, and ICIP series cameras. It exposes the following functionality to the user:

  • Discovery of Network Devices (LAN SCAN)
  • Rapid IP Configuration of Multiple Devices
  • Full Get/Set Access to Device Configuration
  • Factory Reset (ClearConfig) Capabilities
  • Firmware Update Client

Mac OSX Web Plugins

The latest IC Realtime products natively support Mac based web browsers (Safari). Legacy ICR devices beyond 2012 will not support these plugins. For such devices, users should investigate the Mac Client Software (SmartICRSS) instead. Necessary plugins for Safari support are listed below, and as of the time of this writing, there are separate Web Plugins available for the following hardware:

  • IP Camera Web Plugin (stored in \Plugin for IPC\)
  • DVR Web Plugin (stored in \Plugin for Recorder\)
  • NVR Web Plugin (stored in \Plugin for Recorder\)
