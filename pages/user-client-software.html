User/Client Software

Customer facing software. When an IC Realtime product is purchased, these are the applications that are vital for the Enduser to utilize the system. On this KB article, you will see several iterations of the core 'CMS' (Central Monitoring Software) software during it's various stages of evolution. Presently, this article lists these softwares in a 'Newest to Oldest' sort method.

  • IC720 App

    The premier software platform to view and manage Fisheye cameras, and even the latest 360° x 360° cameras from IC Realtime. Further, this application connects to the 'core' lineup of hardware (DVRs, NVRs, and IPCs).


    • Live view and Playback capabilities for ICRealtime Recorders
    • Real 360° control for Fisheye images
    • Immersive viewing with 360° x 360° stitching in realtime
    • Includes normal PTZ controls in addition to advanced VPTZ for 360° cameras

  • SmartICRSS

    IC Realtimes core Central Monitoring (and Management) Software. SmartICRSS is a robust application, suitable for connecting to any IP enabled device in the IC Realtime product lineup. It allows for full usage (View, Playback, and Configuration) of all ICR devices. Multiple monitors are supported, enabling up to (4) screens of Live Viewing capabilities.


    • Discovery of Network Devices (LAN SCAN)
    • Live Viewing
    • Playback
    • System Configuration
    • EMAP
    • VTO/VTH Integration

  • PSS

    The first iteration of the IC Realtimes CMS software - the Pro Surveillance System. This is our legacy Central Monitoring (and Management) Software, which existed prior to it's replacement application, SmartICRSS. PSS is a robust application, suitable for connecting ICR DVRs earlier than 2012. Since the adoption of the new SmartICRSS platform, the legacy PSS software has not been upkept or maintained. There is no long term support for this application any longer.


    • Live View
    • Playback
    • PTZ Control
