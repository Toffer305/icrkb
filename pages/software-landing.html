Software

The central repository for all IC Realtime related Software Knowledge. The purpose of this section is to show the range of software available software via logical categories. Each Software section itself will contain a detailed overview of each release, download links for relevant platforms, and change notes for each release. Related white papers and product documentation are listed as well

User / Client Software

Customer facing software. Applications that are ultimately utilized by the enduser for normal, day-to-day operation of products. These include Central Monitoring Software, Mobile Applications, and Video Player softwares.

Installation Software

Software designed to aid deployment or installation of IC Realtime hardware. These include softwares which will discover IP devices, configure parameters, and necessary packaged required for normal user operation of certain hardware devices.

Device Firmware Updates

The low level code that drives all of our hardware. Firmware releases are listed here and documented as thoroughly as possible. This section will not only provide links to download available updates, but include will explain Hardware Families, Firmware Classes, and provide best efforts for change tracking among different firmware releases.
