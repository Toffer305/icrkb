Tech Tools

A man is only as good as his tools. This section of the Knowledgebase will explain and present the tools that ICRealtime technicians have used, built, and compiled over the years in order to successfully support IC Realtime equipment.

Firmware Matrix

The ultimate compendium for ICRealtime firmware families, and identifying the proper firmware update for your ICRealtime series device. Launches from Google Sheets.

Password Generator

One of the most common calls or issues. Users who have forgotten their password can obtain authorization to receive a temporary "Backdoor" password. Use this page to view the SuperPasswords for today's date. Before releasing the password, always ask yourself: "Am I authorized to release this?"

3rd Party Chart

"Does this WhizzBang 7Q9-ER work with the ICRealtime NVR?" Find out for yourself here, in our detailed report of compatible third party cameras. Please note that while this list is available, there are assuredly cameras which are compatible, but may not yet be included with this list.

HDD Chart

A compatibility chart for Hard Drives. This is used to identify which HDD's can be used with our storage series of products (DVR's, NVR's, AVR's).

CGI Codes

IP Control Codes for ICRealtime Recorders. These are HTTP commands which can be sent to control the devices.

IR Codes

IR Control Codes for ICRealtime Recorders. These are NEC1 IR commands which can be sent to control the devices.

232 Codes

RS-232 Control Codes for ICRealtime Recorders. These are RS-232 commands which can be sent to control the devices.

Mount Diagram

A handy chart which displays which Camera Mounts work with which series of cameras.

Technician Software

Daily use software required for excellence in technical support.
